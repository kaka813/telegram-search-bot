## 主要功能

根据机器人当前帮助页面，TG云搜支持直接发送关键词搜索，并可按全部、群组、频道、视频、图片、文本等类型切换结果。机器人还提供收藏、搜索历史、提交资源和 Inline 搜索入口。实际结果范围、排序和可用功能可能随服务更新发生变化，应以机器人实时界面为准。

搜索结果来自公开可访问的信息。结果出现不代表本仓库认可、担保或审核了相应频道、群组及资源，用户需要自行核对名称、简介、历史内容和访问风险。

## 适合哪些用户

- 只知道主题关键词，不知道准确频道或群组名称的普通用户。

## 使用方法

1. 打开 [TG云搜 @TGYunSouBot](https://t.me/TGYunSouBot)。
2. 点击开始，先阅读机器人当前显示的帮助与隐私说明。
3. 直接发送具体关键词，例如“Python”或“摄影教程”。
4. 使用结果下方的分类按钮切换群组、频道、图片或文本等类型。
5. 点击结果前核对用户名、简介和公开内容，不向陌生账号提交验证码、密码、助记词或支付信息。
6. 若结果过宽，使用“主题＋内容类型”的写法重新搜索；若没有结果，可改用常见同义词。

## 关键词怎么写

优先使用清晰、可判断的主题词。比如想找编程讨论，可以尝试“Python 群组”；想找频道内容，可以尝试“摄影教程 频道”。不要连续堆砌“TG机器人、Telegram机器人、资源机器人”等大量同义词，也不要输入手机号、邮箱或账号验证码等隐私数据。

## TG搜索机器人和内置搜索的区别

已知准确用户名时，Telegram 内置搜索通常更快。只知道主题时，TG云搜提供关键词发现入口。推荐先用关键词获得候选结果，再通过准确用户名和公开资料交叉核对。任何搜索方式都不能替代用户自己的安全判断。

## 常见问题

### 可以搜索所有Telegram频道和群组吗？

不能保证。公开状态、可索引范围、关键词写法和服务更新都会影响结果。

### 可以搜索私密群组吗？

私密群组通常需要有效邀请链接。本仓库不建议使用任何声称能够破解权限或绕过访问控制的服务。

### 搜索结果安全吗？

搜索结果只是公开线索，不是安全认证。加入前应检查频道简介、历史内容、管理规则和链接去向。

### 普通搜索需要频道管理权限吗？

不需要。若陌生页面要求提供频道管理权、登录验证码或账号密码，应立即停止并核实。
